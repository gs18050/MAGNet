# MAGNet
Magnification-Arbitrary Generative Adversarial Network for Image Super-Resolution

Our code is built for google colab.

# Requirements

* Python 3.10
* Pytorch 2.5.1
* PIL 11.0.0
* numpy 1.26.4
* random
* math
* os
* time
* tqdm
